# Maquetación de Página Principal - Tienda Online

Esta tarea consiste en la maquetación visual e interfaz responsiva para la página principal de una tienda online utilizando **Bootstrap 5**.

## Componentes de Bootstrap Utilizados
* **Grid System (`row`, `col-sm-3`, `col-sm-9`, `row-cols-md-3`):** Para la estructuración layout principal responsivo.
* **Carousel (`carousel`):** Ubicado en la parte superior para destacar anuncios y promociones.
* **Cards (`card`):** Para estructurar los elementos visuales de categorías y las tarjetas de productos.
* **Badges y Icons (`badge`, `bootstrap-icons`):** Para simular la notificación visual con contador en el ícono del carrito de compras.
* **Buttons (`btn`, `btn-outline-secondary`):** Botones estilizados para añadir productos.
